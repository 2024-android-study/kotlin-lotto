## 기능 목록
- [x] 로또 구입 금액을 입력받는다.
  - [x] 1000으로 나누어 떨어지지 않을 경우 예외처리한다.
- [x] 쉼표(,)를 기준으로 구분되는 6개의 당첨 번호를 입력 받는다.
- [x] 보너스 번호를 1개 입력 받는다.
- [x] 구입 개수만큼 중복되지 않은 6개의 숫자로 이루어진 로또를 발행한다.
- [ ] 발행한 로또 수량 및 번호를 오름차순으로 출력한다.
- [ ] 사용자가 구매한 로또 번호와 당첨 번호를 비교하여 당첨 내역 및 수익률을 출력한다.
  - [ ] 수익률은 소수점 둘째 자리에서 반올림한다.
  - [ ] 게임을 종료한다.
